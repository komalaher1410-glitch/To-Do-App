# 📋 Taskly – Modern To-Do List Web App

Taskly is a clean and modern **task management web application** built using **HTML, CSS, and JavaScript**.
It helps users organize daily tasks, track productivity, and manage priorities through a simple and responsive interface.

The application uses **LocalStorage**, allowing tasks to remain saved even after refreshing the page.

---

## ✨ Features

* ✅ Add new tasks easily
* ☑️ Mark tasks as completed with checkbox
* 🗑 Delete tasks
* 🎯 Task priority system (Low / Medium / High)
* 📊 Dynamic progress bar showing completed tasks
* 🔎 Task filtering

  * All Tasks
  * Active Tasks
  * Completed Tasks
  * High Priority Tasks
* 🌙 Dark / Light Mode toggle
* 💾 LocalStorage support (tasks saved in browser)
* 🧹 Clear completed tasks
* 🔄 Reset app option
* ⌨️ Keyboard shortcut support
* 🎨 Smooth UI animations
* 📱 Responsive design for desktop and mobile

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the web page
* **CSS3** – Styling, animations, and responsive layout
* **JavaScript (ES6)** – App functionality and DOM manipulation
* **LocalStorage API** – Persistent data storage in browser
